# Dungeon MVP — 数值设计文档

## 行动系统

所有行动有耗时（duration），AV = 反应时 + 耗时，作为单一值入队倒计时。

### 行动定义

| 行动 | 耗时 | 说明 |
|---|---|---|
| 移动 | 300 | 朝相邻地板格走一格 |
| 攻击（近战） | 300 | 朝相邻怪物格子移动时触发，共享移动的耗时 |
| 技能 | 600 | 释放职业技能 |
| 等待 | 800 | 原地消耗一回合 |
| 怪物追击 | 250 | 怪物向玩家移动或近战攻击 |
| 怪物逃跑 | 250 | 怪物朝远离玩家方向移动 |
| 怪物游荡 | 500 | 怪物随机移动 |

### 反应时

从行动确认到实际执行之间的延迟。由敏捷决定：

```
反应时 = max(100 - 敏捷 × 3, 20)
```

| 敏捷 | 反应时 |
|---|---|
| 5 | 85 |
| 10 | 70 |
| 20 | 40 |
| 27+ | 20（下限） |

## 战斗公式

### 近战伤害

```
基础伤害 = max(攻击力 - 目标防御力, 1)
```

攻击力 = 基础攻击 + 装备加成 + Buff 加成（狂暴）

### 暴击

```
暴击判定 = random() < 暴击率
暴击伤害 = 基础伤害 × (1 + 暴击伤害倍率)
```

| 属性 | 玩家基础值 |
|---|---|
| 暴击率 | 5% |
| 暴击伤害 | 50% |

### 技能伤害 / 效果

| 技能 | 效果公式 | MP消耗 |
|---|---|---|
| 火球 | `15 + 法术精通 × 0.5` 伤害，目标须邻接 | 10 |
| 治愈 | `15 + 法术精通 × 1` HP恢复 | 6 |
| 护盾 | 防御 +5，持续3回合 | 5 |
| 狂暴 | 攻击 +5，持续3回合 | 5 |

## 经验系统

### 升级曲线

```
升到下一级所需经验 = 25 × 等级^1.5 + 10 × 等级
```

| 等级 | 升下一级所需经验 |
|---|---|
| 1→2 | 35 |
| 2→3 | 91 |
| 3→4 | 160 |
| 4→5 | 240 |
| 5→6 | 330 |
| 6→7 | 427 |
| 7→8 | 533 |
| 8→9 | 646 |
| 9→10 | 765 |
| 10→11 | 891 |

### 怪物经验

```
老鼠经验 = round(6 + (楼层-1) × 6 × 0.5)
哥布林经验 = round(15 + (楼层-1) × 15 × 0.5)
```

| 楼层 | 老鼠经验 | 哥布林经验 |
|---|---|---|
| 1 | 6 | 15 |
| 2 | 9 | 23 |
| 3 | 12 | 30 |
| 5 | 18 | 45 |
| 10 | 33 | 83 |

### 升级效果

达到经验门槛时自动升级：
- 等级 +1
- 当前经验扣除升级所需，溢出部分保留
- 获得 3 个属性点（待分配）

| 法术精通 | 8 |
| 敏捷 | 10 |
| 暴击率 | 5% |

### HP/MP 成长

```
最大HP = 20 + 等级 × 5 + 防御 × 2
最大MP = 5 + 等级 × 3 + 法术精通
```

### 装备加成

装备提供额外属性，直接累加到战斗数值中：

```
有效攻击 = 基础攻击 + 装备攻击之和 + Buff加成
有效防御 = 基础防御 + 装备防御之和 + Buff加成
```

### 怪物行为

| 行为 | 优先级（数值越低越优先） | 触发条件 |
|---|---|---|
| 逃跑 | 200 | HP < 25% |
| 追击 | 100 | 视野范围内看见玩家 |
| 游荡 | 50 | 无条件（非追击/逃跑时） |
| 等待 | 0 | 无条件（兜底） |

怪物每 tick 检查各行为的冷却和条件，优先执行优先级最高的可用行为。

追击逻辑：
- 与玩家邻接 → 近战攻击（伤害 = 怪物攻击）
- 不与玩家邻接 → 朝玩家方向移动一格

逃跑逻辑：
- 朝远离玩家的方向移动

游荡逻辑：
- 随机选择一个方向移动

## 掉落物系统

### 设计参考

本系统的物品 ID 化、ItemStack、物品注册表和 LootTable 设计参考了 **Minecraft Java Edition 1.16+** 的物品系统架构：

- **Registry 模式**：物品定义存放在 JSON 中，通过 `usize` ID 索引。运行时加载为 `ItemRegistry` 资源，Item 是单例模板不存实例状态
- **ItemStack**：背包中不存完整的物品定义，只存 `(item_id, count)` 对。堆叠上限由物品定义中的 `max_stack` 控制（装备 1，材料 64）
- **LootTable**：怪物挂载 `LootTable` 组件，死亡时掷骰决定掉落。每次 roll 各条目独立判定

### 背包

- 按 `e` 打开背包（双栏界面：左侧背包+装备，右侧地面物品）
- 按 `g` 快速拾取脚下所有物品（不打开背包也可用）
- 相同物品自动堆叠至 `max_stack`
- 装备物品不占背包空间，存储在装备槽中
- `←`/`→` 切换左右栏，`↑`/`↓` 移动选中项
- `Enter` 查看详情，`e` 装备，`d` 丢弃，`u` 卸载装备

## Buff 系统

| Buff | 来源 | 效果 | 持续回合 |
|---|---|---|---|
| 护盾 | 战士/牧师技能 | 防御 +5 | 3 |
| 狂暴 | 战士技能 | 攻击 +5 | 3 |

Buff 每回合结束时倒计时减 1，归零时效果消失。

| 参数 | 值 |
|---|---|
